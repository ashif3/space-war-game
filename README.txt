# space-war-game
This is the game developed by me in python 2.7.
To run this game you need to install python as well as the pygame library.
Here is the tutorial for installing python and pygame: http://webprojects.eecs.qmul.ac.uk/fa303/pgs/install.html

About Game:
The game consists of a single user spaceship and multiple enemy spaceships. All the enemy spaceships have different types of moving and firing strategy, whereas user can move his spaceship and fire using keyboard and mouse. The game has also health bar maintenance feature with good graphics and sound effects.
